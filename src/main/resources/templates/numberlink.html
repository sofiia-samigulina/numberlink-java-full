About the game

The puzzle grid consists of a series of differently-colored number pairs. Your task is to connect each number to its match via a colored path that doesn't

  • cross any other path of
  • double back on itself.
You must also use every empty square on the grid.

How to play?

  1. You should be logged in. Click 'Login'.
  2. Click any number to start drawing a path.
  3. Click any grey tile to set your choose.
  4. If you want play a new game, click 'New game'
